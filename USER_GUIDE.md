# 📘 비디오 자동 생성 프로그램 - 사용 설명서

## 🚀 빠른 시작 (5분이면 시작 가능!)

이 프로그램은 **대본만 입력하면 자동으로 영상을 만들어주는** AI 기반 비디오 생성 도구입니다.

**필요한 것:**
1. Google AI Studio API 키 (무료 시작 가능)
2. BytePlus API 키 (영상 생성용, 무료 크레딧 제공)

---

## 🔑 API 키 발급 방법

### 1️⃣ Google AI Studio API 키 (필수)

**용도:** 대본 분석, 이미지 생성, 음성(TTS) 생성

**발급 방법:**

1. **Google AI Studio 접속**
   - 링크: https://aistudio.google.com/apikey
   - Google 계정으로 로그인

2. **API 키 생성**
   - 화면 상단의 **"Get API key"** 버튼 클릭
   - **"Create API key"** 선택
   - 기존 프로젝트 선택 또는 **"Create API key in new project"** 클릭
   - API 키가 생성됨 (예: `AIzaSyB...`)

3. **API 키 복사**
   - 생성된 키를 복사해두세요 (다시 볼 수 없습니다!)
   - 안전한 곳에 보관

**무료 사용 한도:**
- Gemini 2.5 Flash: 1,000 requests/day
- Imagen 4 Fast: 50 images/day (무료 티어)
- Text-to-Speech (Chirp3 HD): 1M characters/month

**유료 전환 (선택):**

1. **Google Cloud Console 접속**
   - 링크: https://console.cloud.google.com/billing

2. **결제 계정 만들기**
   - 왼쪽 메뉴에서 **"결제" (Billing)** 클릭
   - **"결제 계정 만들기"** 클릭
   - 국가 선택: 대한민국
   - 카드 정보 입력 (체크/신용카드 모두 가능)

3. **프로젝트에 결제 계정 연결**
   - 프로젝트 선택
   - **"결제 계정 연결"** 클릭
   - 방금 만든 결제 계정 선택

**요금:**
- Imagen 4 Fast: $0.02/image (약 ₩29)
- Gemini 2.5 Flash: $0.075/1M tokens (약 ₩109)
- Chirp3 HD: $30/1M characters (약 ₩43,800)

### 2️⃣ BytePlus API 키 (영상 생성용)

**용도:** 이미지를 10초 영상으로 변환 (SeeDance 1.0 Pro Fast)

**발급 방법:**

1. **BytePlus 회원가입**
   - 링크: https://console.byteplus.com/
   - 이메일로 회원가입 (Google/GitHub 로그인 가능)

2. **무료 크레딧 받기**
   - 가입하면 자동으로 **무료 크레딧 2M~5M 토큰** 제공
   - 비디오 생성에도 사용 가능 ✅

3. **ModelArk 접속**
   - 콘솔 왼쪽 메뉴에서 **"ModelArk"** 클릭
   - 링크: https://console.byteplus.com/modelark

4. **SeeDance 모델 활성화**
   - **"Models"** 탭 클릭
   - 검색창에 `seedance` 입력
   - **"seedance-1-0-pro-fast-251015"** 찾기
   - **"Activate"** 버튼 클릭 (무료)

5. **API 키 생성**
   - 왼쪽 메뉴에서 **"API Keys"** 클릭
   - **"Create API Key"** 클릭
   - 키 이름 입력 (예: "video-generation")
   - **"Create"** 클릭
   - API 키 복사 (예: `002c08ea-...`)

**무료 크레딧으로 테스트:**
- 무료 크레딧으로 약 10-20개 영상 생성 가능
- 1개 영상(10초, 720p) = 약 200K 토큰

**유료 전환 (필요시):**

1. **Billing 페이지 접속**
   - 링크: https://console.byteplus.com/billing

2. **충전하기**
   - **"Top up"** 버튼 클릭
   - 최소 충전 금액: $10 (약 ₩14,600)
   - 카드 정보 입력 (Visa, MasterCard, AMEX)

3. **자동 충전 설정 (선택)**
   - 잔액이 $5 이하로 떨어지면 자동 충전
   - 금액 설정 가능

**요금:**
- SeeDance 1.0 Pro Fast (10초, 720p): $0.037/video (약 ₩54)
- 매우 저렴한 편!

## 📖 기본 사용법

### 1단계: API 키 설정

1. 프로그램 실행 후 **"내 페이지"** 클릭
2. **"Gemini API 설정"** 섹션
   - API 키 입력란에 Google AI Studio API 키 붙여넣기
   - 자동으로 유효성 검사됨 (✅ 표시)
3. **"영상화 API 설정"** 섹션
   - BytePlus API 키 입력란에 API 키 붙여넣기

### 2단계: 대본 작성

1. **"대본 입력"** 탭 클릭
2. 대본 입력 (최소 500자 이상 권장)
   ```
   예시:
   안녕하세요! 오늘은 AI가 만드는 영상에 대해 알아보겠습니다.

   먼저, AI는 대본을 분석해서 장면을 나눕니다.
   그 다음, 각 장면에 맞는 이미지를 생성하고...
   ```
3. **"캐릭터 자동 추출"** 버튼 클릭
   - AI가 자동으로 등장인물 분석
   - 필요시 수동으로 수정 가능

### 3단계: 스타일 선택

1. **"스타일 템플릿"** 버튼 클릭
2. 원하는 스타일 선택:
   - 애니메이션 (사이버펑크, 웹툰, 졸라맨 등)
   - 실사 (시네마틱, 다큐멘터리 등)
   - 일러스트 (수채화, 팝아트 등)
   - 전통화 (수묵화, 우키요에 등)
3. **"적용하기"** 클릭

### 4단계: 스토리보드 생성

1. **"스토리보드 생성"** 버튼 클릭
2. AI가 자동으로:
   - 대본을 장면별로 분할 (짧은 대사는 자동 병합)
   - 각 장면에 맞는 이미지 프롬프트 생성
   - 약 1-2분 소요

### 5단계: 이미지 생성

1. **"모든 이미지 생성"** 버튼 클릭
2. Imagen 4 Fast로 자동 생성
   - 1장당 약 5-10초 소요
   - 캐릭터 일관성 유지
3. 마음에 안 드는 이미지는 **"재생성"** 가능

### 6단계: 음성 생성 (TTS)

1. **"나레이션 설정"** 에서 목소리 선택:
   - **Chirp3 HD**: 8가지 한국어 음성 (권장)
   - **Neural2**: 3가지 표준 음성
   - Azure TTS, ElevenLabs (선택)
2. **"모든 오디오 생성"** 버튼 클릭
3. 각 장면마다 자막과 싱크 맞춰서 생성됨

### 7단계: 영상 생성

1. **"영상화 API 설정"**에서 **"영상 생성할 장면 수"** 설정
   - 슬라이더로 조절 (0-180장)
   - 예: 30장 → 처음 30장만 API 영상, 나머지는 줌/패닝 효과
2. **"전체 영상 내보내기"** 버튼 클릭
3. 자동으로:
   - 설정 범위 내: SeeDance API로 10초 영상 생성
   - 설정 범위 밖: 맥락 기반 줌/패닝 효과 (6-15초)
   - 모든 영상 자동 병합
4. 완료되면 MP4 파일 다운로드

- 긴 영상 (20-30분): 30% API + 70% 효과

### 자막 커스터마이징

1. **"자막 설정"** 클릭
2. 프리셋 선택 또는 커스텀:
   - 글자 크기, 색상, 외곽선
   - 배경 박스 (불투명도 조절)
   - 위치 (상단/중앙/하단)
3. 설정은 자동 저장됨

1. **"나만의 화풍 등록"** 클릭
2. 참고 이미지 3개 이상 업로드
3. 화풍 이름 입력
4. **"저장하기"**
5. 다음부터 **"내 그림체"** 탭에서 선택 가능

## 🔧 문제 해결 (FAQ)

### Q1: "API 키가 유효하지 않습니다"
- API 키를 다시 복사해서 붙여넣기
- Google AI Studio에서 키가 활성화되어 있는지 확인
- 결제 계정이 연결되어 있는지 확인 (유료 기능 사용 시)

### Q2: "BytePlus 모델을 찾을 수 없습니다"
- ModelArk에서 `seedance-1-0-pro-fast-251015` 모델 활성화 필요
- 모델 검색 → Activate 버튼 클릭

### Q3: "크레딧이 부족합니다"
- BytePlus 콘솔에서 잔액 확인
- 무료 크레딧을 모두 사용한 경우 → Top up ($10 이상 충전)
- Google AI Studio 무료 한도 초과 → 결제 계정 연결

### Q4: "이미지가 일관성이 없어요"
- 캐릭터 설정에서 **"시각적 묘사"** 더 자세히 작성
- 참고 이미지 업로드 (캐릭터 카드에서 이미지 추가)
- 스토리보드 재생성 시 캐릭터 일관성 강화됨

### Q5: "영상 생성이 너무 느려요"
- 정상입니다! SeeDance API는 1장당 약 1-2분 소요
- 백그라운드에서 작동하므로 기다리세요
- 영상 생성 범위를 줄이면 더 빠름 (예: 30장만)

### Q6: "음성이 자막과 안 맞아요"
- 자동으로 싱크 맞춤 (자막 길이 기준 6-15초)
- TTS 속도 조절: 나레이션 설정에서 **"목소리 속도"** 조정 (0.5x~2.0x)

## 🎉 팁과 트릭

### 1. 대본 작성 팁
- 장면 전환을 명확히: "한편, 사무실에서는..."
- 감정 표현 활용: "!" (긴박감), "?" (의문), "..." (여운)
- 방향 키워드: "왼쪽에서", "위로", "다가오며" → 자동 패닝 효과

### 2. 비용 최적화
- 대본을 먼저 완성하고 한 번에 생성 (재생성 최소화)
- 영상 생성 범위는 처음 30-50%만
- 짧은 테스트는 무료 크레딧으로 충분

### 3. 품질 향상
- 캐릭터 초상화 이미지 등록 (일관성 UP)
- 스타일 템플릿 신중히 선택
- 이미지 프롬프트 수동 수정 가능 (고급 사용자)

### 4. 시간 절약
- 스토리보드 생성 → 이미지 생성 → TTS 생성 순서대로
- 병렬 작업: 이미지 생성 중에 다른 프로젝트 준비
- 자동 저장 기능 활용 (localStorage)
