## 이 파일 보는 방법
- 프로젝트 폴더의 "사용설명서.md" 파일을 메모장이나 텍스트 에디터로 열기
- 또는 GitHub 저장소에서 온라인으로 보기: https://github.com/Breaduck/google-youtubeproject/blob/브랜치2/사용설명서.md

## 프로그램 소개

이 프로그램은 대본만 입력하면 자동으로 영상을 만들어주는 AI 기반 도구입니다.

필요한 것:
1. Google AI Studio API 키 (이미지 생성, 음성 생성용)
2. BytePlus API 키 (영상 생성용)

두 가지 모두 무료로 시작할 수 있습니다.

---

## 1단계: Google AI Studio API 키 발급

### 1-1. API 키 생성하기

1. 브라우저를 열고 https://aistudio.google.com/apikey 로 접속합니다

2. 구글 계정으로 로그인합니다
   - Gmail 계정이 있으면 그대로 로그인
   - 없으면 새로 만들기

3. 화면 상단에 "Get API key" 라는 파란색 버튼이 있습니다. 이것을 클릭합니다

4. 두 가지 선택지가 나옵니다:
   - "Create API key in new project" (새 프로젝트에 키 만들기) - 처음 사용하는 경우 이것 선택
   - "Create API key in existing project" (기존 프로젝트에 키 만들기) - 이미 프로젝트가 있으면 선택

5. "Create API key in new project"를 클릭합니다

6. 잠시 기다리면 API 키가 생성됩니다
   - 예시: AIzaSyB4TZb33S2udfTEyBFFIcQ3DyfUCvhSK4s
   - 이런 형태의 긴 문자열입니다

7. "Copy" 버튼을 클릭해서 API 키를 복사합니다

8. 복사한 키를 메모장에 붙여넣어서 안전하게 보관합니다
   - 중요: 이 키는 다시 볼 수 없으니 반드시 저장하세요
   - 다른 사람에게 절대 공유하지 마세요

### 1-2. Text-to-Speech (TTS) API 활성화하기

나레이션 음성을 생성하려면 추가로 TTS API를 활성화해야 합니다.

1. 브라우저에서 https://console.cloud.google.com 으로 접속합니다

2. 구글 계정으로 로그인합니다 (위에서 사용한 계정과 동일)

3. 화면 왼쪽 위에 프로젝트 이름이 표시됩니다
   - 예: "My Project 12345"
   - 방금 만든 프로젝트가 자동으로 선택되어 있습니다

4. 화면 위쪽 중앙에 검색창이 있습니다
   - "검색 제품 및 리소스" 라고 적혀있는 곳

5. 검색창에 "text to speech" 라고 입력합니다

6. 검색 결과에서 "Cloud Text-to-Speech API" 를 클릭합니다

7. "사용" 또는 "ENABLE" 버튼이 보입니다. 이것을 클릭합니다

8. 잠시 기다리면 API가 활성화됩니다

9. 똑같은 방법으로 "speech to text"도 검색해서 활성화합니다
   - 검색창에 "speech to text" 입력
   - "Cloud Speech-to-Text API" 클릭
   - "사용" 버튼 클릭

### 1-3. 무료 사용 한도

Google AI Studio는 처음에 무료로 사용할 수 있습니다:

- 이미지 생성 (Imagen 4 Fast): 하루 50장까지 무료
- 음성 생성 (Chirp3 HD): 한 달에 100만 글자까지 무료
- 대본 분석 (Gemini): 하루 1,000번까지 무료

무료 한도를 모두 사용하면 더 이상 생성되지 않습니다.
계속 사용하려면 유료로 전환해야 합니다.

### 1-4. 유료로 전환하기 (선택사항)

무료 한도가 부족하면 카드를 등록해서 유료로 사용할 수 있습니다.

1. https://console.cloud.google.com/billing 으로 접속합니다

2. 왼쪽 메뉴에서 "결제" 또는 "Billing"을 클릭합니다

3. "결제 계정 만들기" 버튼을 클릭합니다

4. 국가를 선택합니다
   - "대한민국" 선택

5. 카드 정보를 입력합니다
   - 체크카드, 신용카드 모두 가능
   - Visa, Mastercard, AMEX 사용 가능

6. "제출" 버튼을 클릭합니다

7. 다시 프로젝트 페이지로 돌아옵니다

8. "결제 계정 연결" 버튼을 클릭합니다

9. 방금 만든 결제 계정을 선택합니다

10. "계정 설정" 버튼을 클릭합니다

이제 무료 한도를 초과해도 자동으로 과금됩니다.

요금표:
- 이미지 1장: 0.02달러 (약 29원)
- 음성 100만 글자: 30달러 (약 44,000원)
- 대본 분석 100만 토큰: 0.075달러 (약 109원)

## 2단계: BytePlus API 키 발급

### 2-1. 회원가입하기

1. 브라우저에서 https://console.byteplus.com 으로 접속합니다

2. 화면 오른쪽 위에 "Sign Up" 버튼을 클릭합니다

3. 이메일 주소를 입력합니다
   - Gmail, Naver, Daum 등 아무 이메일이나 가능

4. 비밀번호를 설정합니다
   - 최소 8자 이상
   - 영문, 숫자, 특수문자 포함

5. "Sign Up" 버튼을 클릭합니다

6. 이메일로 인증 메일이 발송됩니다

7. 이메일을 열고 "Verify Email" 버튼을 클릭합니다

8. 회원가입 완료!

### 2-2. 무료 크레딧 받기

회원가입을 하면 자동으로 무료 크레딧이 제공됩니다:
- 2백만 ~ 5백만 토큰 (계정마다 다름)
- 비디오 생성에 사용 가능
- 약 10~20개 영상을 무료로 만들 수 있습니다

### 2-3. SeeDance 모델 활성화하기

영상을 만들려면 SeeDance 모델을 먼저 활성화해야 합니다.

1. https://console.byteplus.com/modelark 로 접속합니다

2. 왼쪽 메뉴에서 "Models" 탭을 클릭합니다

3. 화면 위쪽에 검색창이 있습니다

4. 검색창에 "seedance" 라고 입력합니다

5. 검색 결과에서 "seedance-1-0-pro-fast-251015" 를 찾습니다

6. 오른쪽에 "Activate" 버튼이 있습니다. 이것을 클릭합니다

7. 확인 창이 나오면 "Confirm" 버튼을 클릭합니다

8. 잠시 기다리면 "Activated" 상태로 변경됩니다

### 2-4. API 키 생성하기

1. 왼쪽 메뉴에서 "API Keys" 를 클릭합니다

2. 화면 오른쪽 위에 "Create API Key" 버튼을 클릭합니다

3. API 키 이름을 입력합니다
   - 예: "video-generation" 또는 "영상생성용"
   - 나중에 구분하기 쉬운 이름으로 작성

4. "Create" 버튼을 클릭합니다

5. API 키가 생성됩니다
   - 예시: 002c08ea-0f36-4ad1-84ef-25740c463f5d
   - 이런 형태의 문자열입니다

6. "Copy" 버튼을 클릭해서 복사합니다

7. 복사한 키를 메모장에 붙여넣어서 안전하게 보관합니다
   - 중요: 다시 볼 수 없으니 반드시 저장하세요

### 2-5. 유료로 전환하기 (선택사항)

무료 크레딧을 모두 사용하면 추가 충전이 필요합니다.

1. https://console.byteplus.com/billing 로 접속합니다

2. "Top up" 버튼을 클릭합니다

3. 충전 금액을 선택합니다
   - 최소 10달러 (약 14,600원)
   - 추천: 20달러 충전하면 약 370개 영상 생성 가능

4. 카드 정보를 입력합니다
   - Visa, Mastercard, AMEX 사용 가능

5. "Pay" 버튼을 클릭합니다

6. 결제 완료!

요금표:
- 10초 영상 1개 (720p): 0.037달러 (약 54원)

## 3단계: 프로그램에서 API 키 설정하기

### 3-1. 프로그램 접속

1. 브라우저에서 https://branch2-9jl.pages.dev 로 접속합니다

2. 화면에 "내 프로젝트" 라는 제목이 보입니다

3. 화면 오른쪽 위에 "내 페이지" 버튼을 클릭합니다

### 3-2. Google AI Studio API 키 입력

1. "Gemini API 설정" 섹션을 찾습니다

2. "API 키" 입력란에 1단계에서 복사한 Google AI Studio API 키를 붙여넣습니다
   - Ctrl+V 또는 오른쪽 클릭 > 붙여넣기

3. 자동으로 유효성 검사가 진행됩니다

4. 초록색 체크 표시와 "유효함" 이라는 메시지가 나오면 성공입니다

5. 빨간색 X 표시가 나오면 키가 잘못되었으니 다시 확인하세요

### 3-3. BytePlus API 키 입력

1. 아래로 스크롤해서 "영상화 API 설정" 섹션을 찾습니다

2. "BytePlus API 키" 입력란에 2단계에서 복사한 API 키를 붙여넣습니다

3. 입력하면 자동으로 저장됩니다

4. 완료!

## 4단계: 영상 만들기 (기본 사용법)

### 4-1. 새 프로젝트 만들기

1. 화면 왼쪽 위의 "홈" 버튼을 클릭합니다

2. "내 프로젝트" 화면이 나옵니다

3. 화면 중앙의 텍스트 입력란에 대본을 작성합니다
   - 최소 500자 이상 권장
   - 예시:
   ```
   안녕하세요! 오늘은 AI가 자동으로 영상을 만드는 방법을 알아보겠습니다.

   먼저, AI는 대본을 분석해서 장면을 나눕니다.
   각 장면마다 어떤 이미지가 필요한지 판단하죠.

   그 다음, 이미지를 생성합니다.
   Imagen 4라는 AI가 텍스트를 읽고 그림을 그려줍니다.

   마지막으로 음성을 합성합니다.
   Chirp3라는 AI가 대본을 읽어주는 거죠.

   이렇게 만들어진 이미지와 음성을 합쳐서 영상이 완성됩니다!
   ```

4. 대본을 다 쓰면 "프로젝트 시작하기" 버튼을 클릭합니다

5. AI가 자동으로 대본을 분석합니다
   - 약 10~30초 소요
   - 등장인물을 자동으로 찾아냅니다
   - 제목도 자동으로 생성합니다

6. 분석이 끝나면 캐릭터 설정 화면으로 이동합니다

### 4-2. 캐릭터 설정하기

1. AI가 자동으로 찾은 캐릭터 목록이 표시됩니다

2. 각 캐릭터의 "시각적 묘사"를 확인합니다
   - 예: "젊은 여성, 긴 검은 머리, 안경 착용, 정장 차림"
   - 마음에 안 들면 수정할 수 있습니다

3. 캐릭터 일관성을 위해 참고 이미지를 업로드할 수 있습니다 (선택사항)
   - "이미지 업로드" 버튼 클릭
   - 컴퓨터에서 이미지 파일 선택
   - AI가 이 이미지를 참고해서 비슷하게 그립니다

4. 수정이 끝나면 "다음 단계" 버튼을 클릭합니다

### 4-3. 스타일 선택하기

1. "스타일 템플릿" 버튼을 클릭합니다

2. 원하는 스타일을 선택합니다
   - 애니메이션: 사이버펑크, 웹툰, 졸라맨 등
   - 실사: 시네마틱, 다큐멘터리, 브이로그 등
   - 일러스트: 수채화, 팝아트, 연필 스케치 등
   - 전통화: 수묵화, 우키요에, 민화 등

3. 썸네일을 클릭하면 선택됩니다

4. "적용하기" 버튼을 클릭합니다

### 4-4. 스토리보드 생성하기

1. "스토리보드 생성" 버튼을 클릭합니다

2. AI가 자동으로 작업합니다:
   - 대본을 장면별로 나눔 (짧은 대사는 자동으로 합침)
   - 각 장면마다 이미지 프롬프트 생성
   - 감정 강도, 효과 설정
   - 약 1~2분 소요

3. 완료되면 장면 목록이 표시됩니다
   - 예: "장면 1", "장면 2", "장면 3" ...
   - 각 장면의 대사와 이미지 설명이 보입니다

4. 마음에 안 드는 장면은 클릭해서 수정할 수 있습니다

### 4-5. 이미지 생성하기

1. "모든 이미지 생성" 버튼을 클릭합니다

2. AI가 자동으로 모든 장면의 이미지를 생성합니다
   - 1장당 약 5~10초 소요
   - 100장이면 약 10~15분 걸립니다

3. 생성 중에는 진행률이 표시됩니다
   - "23/100 - 23% 완료"

4. 완료되면 각 장면에 이미지가 표시됩니다

5. 마음에 안 드는 이미지는 클릭해서 "재생성" 버튼을 누르면 다시 만들어집니다

### 4-6. 음성 생성하기 (TTS)

1. "모든 오디오 생성" 버튼을 클릭합니다

2. AI가 자동으로 각 장면의 대사를 읽어줍니다
   - 음성은 "내 페이지"의 "나레이션 설정"에서 선택한 것으로 생성됩니다
   - 기본값: Chirp3 HD - Kore (활기찬 여성 목소리)

3. 1장당 약 2~5초 소요

4. 완료되면 각 장면에 재생 버튼이 나타납니다
   - 클릭하면 음성을 들어볼 수 있습니다

### 4-7. 영상 생성 및 다운로드

1. "전체 영상 내보내기" 버튼을 클릭합니다

2. AI가 자동으로:
   - 설정한 범위 내의 장면은 SeeDance API로 10초 영상 생성
   - 설정 범위 밖의 장면은 줌/패닝 효과 자동 적용 (무료)
   - 모든 영상을 하나로 합침

3. 영상 생성은 시간이 오래 걸립니다
   - 1장당 약 1~2분
   - 30장이면 약 30~60분 소요
   - 백그라운드에서 진행되므로 기다리세요

4. 완료되면 자동으로 MP4 파일이 다운로드됩니다

5. 파일을 열어서 확인하세요!

## 5단계: 설정 커스터마이징 (고급)

### 5-1. 영상 생성 범위 설정하기

비용을 절약하려면 일부 장면만 API로 생성하고 나머지는 효과로 처리할 수 있습니다.

1. "내 페이지" 버튼 클릭

2. "영상화 API 설정" 섹션 찾기

3. "영상 생성할 장면 수" 슬라이더 조절
   - 0~180장 범위
   - 예: 30장으로 설정하면 처음 30장만 API 영상, 나머지는 줌/패닝 효과

4. 숫자 입력란에 직접 입력도 가능

5. 자동으로 저장됨

권장 설정:
- 짧은 영상 (5분 이하): 전체 API 생성
- 중간 영상 (10분): 50% API + 50% 효과
- 긴 영상 (20분 이상): 30% API + 70% 효과

### 5-2. 목소리 변경하기

1. "내 페이지" 버튼 클릭

2. "나레이션 설정" 섹션 찾기

3. "목소리 모델" 에서 선택:
   - Chirp3 HD: 8가지 한국어 음성 (권장)
   - Neural2: 3가지 표준 음성
   - Azure TTS: 4가지 음성
   - ElevenLabs: 커스텀 음성 (별도 가입 필요)

4. Chirp3 HD를 선택했다면 아래에서 구체적인 목소리 선택:
   - Kore: 활기찬, 명랑한 (여성)
   - Aoede: 차분한, 우아한 (여성)
   - Leda: 중립적, 안정적 (여성)
   - Zephyr: 경쾌한, 산뜻한 (여성)
   - Charon: 중후한, 깊은 (남성)
   - Fenrir: 힘찬, 강렬한 (남성)
   - Puck: 경쾌한, 활발한 (남성)
   - Orus: 차분한, 부드러운 (남성)

5. 목소리 속도도 조절 가능
   - 슬라이더로 0.5배 (느리게) ~ 2.0배 (빠르게) 조절

6. 자동으로 저장됨

### 5-3. 자막 스타일 변경하기

1. "내 페이지" 버튼 클릭

2. "자막 설정" 섹션 찾기

3. 프리셋 선택 또는 커스텀:
   - default-white: 흰색 텍스트 (기본)
   - default-yellow: 노란색 텍스트
   - black-outline: 검은 테두리
   - bg-box: 배경 박스 있음
   - custom: 커스터마이징

4. 세부 설정:
   - 글자 크기: 16~64
   - 글씨체: Pretendard, Noto Sans 등
   - 색상: 색상 선택기로 변경
   - 외곽선: 두께, 색상 설정
   - 배경 박스: 색상, 불투명도
   - 위치: 상단/중앙/하단

## 자주 묻는 질문 (FAQ)

질문 1: API 키를 입력했는데 "유효하지 않습니다" 라고 나옵니다.

답변:
- API 키를 다시 복사해서 붙여넣어보세요
- 앞뒤에 공백이 있으면 안 됩니다
- Google AI Studio에서 키가 삭제되지 않았는지 확인하세요
- 유료 기능을 사용하려면 결제 계정을 연결해야 합니다

---

질문 2: BytePlus에서 "ModelNotOpen" 오류가 나옵니다.

답변:
- ModelArk에서 SeeDance 모델을 활성화해야 합니다
- 위의 "2-3. SeeDance 모델 활성화하기" 부분을 다시 확인하세요
- 모델 이름: seedance-1-0-pro-fast-251015

---

질문 3: 크레딧이 부족하다고 나옵니다.

답변:
- BytePlus 콘솔에서 잔액을 확인하세요
- 무료 크레딧을 모두 사용했으면 Top up 으로 충전하세요
- Google AI Studio는 무료 한도 초과 시 결제 계정을 연결하세요

---

질문 4: 이미지가 장면마다 다르게 나옵니다 (일관성 없음).

답변:
- 캐릭터 설정에서 "시각적 묘사"를 더 자세히 작성하세요
- 참고 이미지를 업로드하면 일관성이 높아집니다
- 스토리보드를 재생성해보세요

---

질문 5: 영상 생성이 너무 오래 걸립니다.

답변:
- 정상입니다. SeeDance API는 1장당 1~2분 소요됩니다
- 백그라운드에서 작동하므로 그냥 기다리세요
- 영상 생성 범위를 줄이면 더 빠릅니다 (예: 30장만)

---

질문 6: 음성이 자막과 싱크가 안 맞습니다.

답변:
- 자동으로 자막 길이에 맞춰서 싱크를 맞춥니다
- TTS 속도를 조절해보세요: "나레이션 설정"에서 0.5x~2.0x
- 너무 긴 대사는 짧게 나눠보세요
